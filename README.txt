Per compilare il progetto è necessario utilizzare tutti i file all'interno della cartella "Project", 
fatta eccezione per "info.txt" che contiene informazioni utili sui datasets utilizzati.

Nel file "decisiontreelearner.py" è stata utilizzata la keyword "nonlocal" 
supportata solo da Python 3 in poi.

Per mostrare il plot degli errori è stata utilizzata la libreria matplotlib.

Il file su cui eseguire il run è "testdataset.py". All'interno del modulo è fornita una spiegazione 
dettagliata che descrive i possibili output per ogni dataset:

- Stampare un plot degli errori su training set e test set in funzione degli errori 
- Stampare l'albero su cui non è stato eseguito il pre-pruning

La cartella "Images" contiene i tre grafici ottenuti dai tre dataset utilizzati.
